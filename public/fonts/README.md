Place your Segoe UI font files in this folder so the app can load them via /fonts/ paths.

Actual filenames in this project (examples):
- Segoe UI.ttf              (regular)
- Segoe UI Bold.ttf         (bold)
- Segoe UI Italic.ttf       (italic)
- Segoe UI Bold Italic.ttf  (bold italic)

Recommended formats: provide at least `woff2` and `woff` for broad browser support.

After copying the files here (or if they already exist), hard-refresh the browser (Ctrl+Shift+R) or open the app in Incognito to ensure the new local fonts are loaded.
